HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags dictate the type of element.
  • An element is all of the information contained between parent tags.
  • It's important to use the alt tag to include a short description of an image. This will help with screen readers as well as SEO.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • starting with a . indicates a class selector e.g. .card

Git

  • Check Git Branch - "git status"
  • Open SSH Agent - "ssh-agent"
  • Switch Git Branch - "git checkout "branch name e.g. main""
  • Create New Git Branch - "git checkout -b feature/"relevant title e.g. add-html""
  • Add changed files "git add -A"
  • Commit changes - git commit -m "sumamry note e.g. added code to HTML file" (-m flag adds a message)
  • Git fLow - add, commit, push, pull

JavaScript

  • JavaScript uses the "const" keyword to declare immutable variables
  • Javascript uses the "let" keyword to declare a mutable variable
  • Javascript uises the "function" keyword to declare a function
